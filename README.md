# 2DPingPong
2D ping pong in Unity 5.5 2017.3.0f3


#### Play with Ping Pong

* Player 1 (left): use _s_ and _w_ to control player

* Player 2 (right): use arrow key ⬆︎ and ⬇︎ to control player
